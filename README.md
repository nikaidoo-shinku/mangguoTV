### 1、**项目介绍**：

- 基于 vue2 开发的仿芒果 TV 的 h5 前端项目。基本功能包括：顶部分类、视频分集、视频进度缓存、图片懒加载、观看历史、用户评论、点赞、收藏视频、搜索、登录页、个人主页等。

### 2、**技术栈**：

- 前端：vue2 + js、scss、Vue Router、vuex、axios;
- vue 项目基于@vue/cli 5.0.8 搭建;
- UI 组件库：vant、swiper;
- 后端数据：使用 fask mock 模拟后端 api 数据;
